#  Crisbee OS

**Crisbee OS** is a free, secure, AI-native operating system built on Linux,
powered by local LLMs and designed for privacy-first human–computer interaction.

##  Vision
- AI as the primary OS interface
- 100% local & offline LLM support
- Security-by-design
- Open-source & free forever

##  Core Features (Planned)
- Natural language system control
- Local LLM (Qwen, Mistral, LLaMA)
- Secure intent-based execution
- AI-powered desktop shell

##  Status
**v0.1 – AI Core (CLI prototype)**

##  Tech Stack
- Ubuntu LTS
- Python
- Ollama (local LLM)
- Qt (UI – upcoming)
